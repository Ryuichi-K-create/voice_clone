# ファインチューニング用 録音原稿

## 注意事項

- 静かな環境で録音すること
- マイクから20~30cm離れて、一定の距離を保つ
- 自然な速度で、はっきりと読む
- 各文の間に**1秒程度の間（ま）**を空ける
- 途中で詰まったら、その文の最初から読み直せばOK
- 全文で約15〜20分（100文）
